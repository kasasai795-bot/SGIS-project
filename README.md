# Smart Grievance System (SGIS)

## Description

This project is a Smart Grievance System that allows users to submit complaints and track their status. It includes user dashboard and admin handling.

## Features

* Submit complaints
* Track complaint status
* User dashboard
* Admin management

## Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Node.js (Express)
* Database: (mention if used, else write "Not used / Local storage")

## How to Run (Local)

1. Download or clone this repository

2. Go to project folder:
   cd SGIS_fixed

3. Install dependencies:
   npm install

4. Start backend:
   node server.js

5. Open frontend:
   Open HTML files in browser http://localhost:8080/login.html
